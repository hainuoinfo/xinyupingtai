  •  生成不超过指定大小的缩略图
  •  生成固定大小的缩略图

(无需保存设置即可预览)
缩略图质量:
设置图片附件缩略图的质量参数，范围为 0～100 的整数，数值越大结果图片效果越好，但尺寸也越大
缩略图大小:
X 设置缩略图的大小，小于此尺寸的图片附件将不生成缩略图
水印:
不启用水印功能
#1 #2 #3
#4 #5 #6
#7 #8 #9
您可以设置自动为用户上传的 JPG/PNG/GIF 图片附件添加水印，请在此选择水印添加的位置(3x3 共 9 个位置可选)。不支持动画 GIF 格式
(无需保存设置即可预览)
水印添加条件:
X 小于此尺寸的图片附件将不添加水印，0 以水印图片大小为添加条件
水印图片类型:
  •  GIF 类型水印
  •  PNG 类型水印
  •  文本类型水印
 如果设置 GIF 类型的文件作为水印，水印图片为 ./images/common/watermark.gif，如果设置 PNG 类型的文件作为水印，水印图片为 ./images/common/watermark.png，您可替换水印文件以实现不同的水印效果。如果设置文本类型的水印并且使用 GD 图片处理库，那么还需要 FreeType 库支持才能使用
水印融合度:
设置 GIF 类型水印图片与原始图片的融合度，范围为 1～100 的整数，数值越大水印图片透明度越低。PNG 类型水印本身具有真彩透明效果，无须此设置。本功能需要开启水印功能后才有效
JPEG 水印质量:
设置 JPEG 类型的图片附件添加水印后的质量参数，范围为 0～100 的整数，数值越大结果图片效果越好，但尺寸也越大。本功能需要开启水印功能后才有效
文本水印文字:
如果您指定的 TrueType 字体为中文字体文件，那么您可以在文本水印中书写中文
双击输入框可扩大/缩小
文本水印 TrueType 字体文件名
填写存放在 ./images/fonts/ch 或 ./images/fonts/en 目录下的 TTF 字体文件，支持中文字体。如使用中文 TTF 字体请使用包含完整中文汉字的字体文件。
文本水印字体大小:
设置文本水印字体大小，请按照字体设置相应的大小
文本水印显示角度:
0 度为从左向右阅读文本
文本水印字体颜色

文本水印阴影纵向偏移量:
设置文本水印阴影纵向偏移量，此数值不宜设置的太大
文本水印阴影颜色
输入 16 进制颜色代表文本水印阴影字体颜色
文本水印横向偏移量(ImageMagick)
设置水印文本输出后向屏幕中央的横向的偏移值。本设置只适用于 ImageMagick 图片处理库

设置水印文本纵向的倾斜角度。本设置只适用于 ImageMagick 图片处理库

远程附件目录的绝对路径或相对于 FTP 主目录的相对路径，结尾不要加斜杠“/”，“.”表示 FTP 主目录

支持 HTTP 和 FTP 协议，结尾不要加斜杠“/”；如果使用 FTP 协议，FTP 服务器必需支持 PASV 模式，为了安全起见，使用 FTP 连接的帐号不要设置可写权限和列表权限
FTP 传输超时时间:

 注意：如果选择“只允许上传到远程服务器”，当尝试上传附件至远程服务器失败时，将放弃上传附件并提示错误信息。
允许的附件扩展名:

双击输入框可扩大/缩小
禁止的附件扩展名:
禁止这些扩展名结尾的附件使用远程附件功能，每行一个，不区分大小写，留空为不限制。
双击输入框可扩大/缩小
附件尺寸下限:
单位：KB，只有尺寸大于当前设置的附件才使用远程附件功能，0 或留空为不限制。
 
附件链接有效期
单位：小时，0 或留空表示附件链接永久有效，过期后链接自动更新。本功能可有效防止附件被盗链或附件被软件批量下载，但是会给会员的正常下载带来不便
下载附件来路检查:
  •  是
  •  否
 选择“是”将检查下载附件的来路，来自其他网站或论坛的下载请求将被禁止。注意: 本功能在开启“帖子中显示图片附件”时，会加重服务器负担
隐藏远程附件真实路径:
  •  是
  •  否
 选择是，将加重论坛服务器负担，并明显增加论坛服务器流量；选择否，下载的附件与上传的附件文件名将会不一致。
